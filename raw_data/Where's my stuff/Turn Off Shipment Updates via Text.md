# Turn Off Shipment Updates via Text for a Specific Package



You can turn off Shipment Updates via Text for a specific package at any time.



To turn off Shipment Updates via Text for a specific package:



1. Go to Your Orders.
2. Select Track your package for the package for which you want to turn off shipment updates via text. You'll see the option to turn off Shipment Updates via Text for that package.



Note: You may still receive a text message after you unsubscribe, if we sent text messages that haven't reached you.
