# Add a Shopping Profile

An adult shopping profile is automatically created when an account is opened. You can create
up to five additional profiles for one account.

You can add up to five shopping profiles to personalize your experience:

1. Go to the
   Accounts & Lists menu and select
   Manage Profiles.
2. Select
   Add Profile.
   - Amazon Profiles displays profiles already created in your account from other services
     like Prime Video, Alexa, or Amazon Family. Using the same profile for shopping allows better recommendations based on all your
     Amazon activities.
   - If the profile you want to use is listed, you can select it and start shopping. If
     not, proceed to the next step.
3. Select
   Create profile.
4. Give your new profile a name and optional photo. The name can't be the same as any
   other profiles within your account.
5. Select
   Continue.
   - Once you've added a new profile, you're automatically switched to this profile and
     ready to shop.
   - You can edit your profiles through the
     View option.

**Note:**

- You can have six active profiles (one automatically created for the account holder,
  plus five more). If you reach the limit, you can remove a profile anytime. Select
  Remove profile and follow the on-screen prompts.
- In certain jurisdictions, a user that is 13-17 years of age may be prompted to create
  a Teen profile when they log in to an existing Amazon account. Unless prompted, you
  can't create a teen shopping profile.
