# Return Windows and Eligibility Exceptions

## Standard Return Window

Most items sold on Amazon.com can be returned within 30 days of delivery.

## 7-Day Return Window

Applicable to:
- Kindle books accidentally purchased from the Kindle Store that have not been read
- Digital textbooks, workbooks, and educational content that have not been downloaded
- Songs or albums accidentally purchased using Alexa from the Digital Music Store

## 15-Day Return Window

Applicable to:
- Apple brand products sold in new condition
- Boost Infinite brand products sold in new condition
- Amazon Haul items priced above $3

Notes:
- AppleCare+ cancellations follow plan terms
- Amazon Haul items priced at $3 or less are non-returnable and non-refundable

## 90-Day Return Window

Applicable to:
- Select Amazon Renewed items in Acceptable, Good, or Excellent condition
- Most nonperishable baby products
- Amazon Birthday Registry purchases made by someone other than the registry owner
- Custom Gift List purchases
- Mattresses excluding crib mattresses

## 180-Day Return Window

Applicable to:
- Amazon Wedding Registry purchases made by someone other than the registry owner

## 365-Day Return Window

Applicable to:
- Amazon Renewed Premium products
- Amazon Baby Registry purchases made by someone other than the registry owner
