# Initiating, Packaging, and Sending Returns

## Starting a Return

To initiate a return:
1. Go to Your Orders
2. Select “Return Items” beside the eligible item

## Return Drop-Off Options

Amazon supports:
- Over 8,000 return locations
- Free returns for eligible items
- Box-free returns
- Label-free returns

Most return locations are within a 5-mile radius of customer addresses.

## Packaging Requirements

Returned items should include:
- Original manufacturer packaging
- Tags
- Accessories
- Components
- Manuals
- Certificates of authenticity
- Inserts

Additionally:
- Hygiene seals and liners must remain intact
- Items should be unused or in original condition

## Return Deadlines

Items must be returned before the “return by date” shown in:
- Return confirmation emails
- Return Request section in Order History

## Important Return Rules

Customers should:
- Follow return instructions for each item
- Avoid combining items from separate orders in the same package

## Personal Information Removal

Before returning electronics or devices containing personal information:
- Completely erase stored data
- Follow manufacturer instructions

Examples include:
- Computers
- Phones
- Electronics

## Refund Reversal Conditions

If a refund is issued before the item is returned:
- Customers may be charged if the item is not sent back

Charges are reversed once Amazon processes the returned item.

## Hazardous Item Returns

Defective hazardous items must be returned through instructions provided in the Online Returns Center.
