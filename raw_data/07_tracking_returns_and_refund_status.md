# Tracking Returns and Refund Status

## Tracking a Return

Customers can track returns using:
- Return confirmation emails
- Carrier tracking numbers
- Your Returns section in their Amazon account

## Return Tracking Through Your Returns

To track a return:
1. Go to Your Returns
2. Locate the item under Last 3 Months
3. Review return and refund details

## Return Statuses

### Return Requested on [DATE]

Amazon accepted the return request and is waiting to receive the item.

### Refund Issued

Amazon processed the return and issued a refund.

### Item No Longer Eligible for Return

The item exceeded the return window.

## Refund Status Checks

Customers can also check refund status using:
- Return/Refund Status links in confirmation emails
- Your Returns section

## Refund Visibility

Once refunds are issued:
- Refund details become visible
- Customers receive refund summary emails
- Transactions appear in Your Transactions

## Gift Return Refund Distribution

### Gift Recipient Returns

Refunds are issued to:
- Amazon gift card balance of the account used to create the return request

### Gift Purchaser Returns

Refunds are issued to:
- The refund method selected in the Online Returns Center
