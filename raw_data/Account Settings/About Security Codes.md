# About Security Codes

To protect your account and sensitive information, some Amazon services require additional
verification through a security code.

**If you haven’t received your code:**

1. Verify your mobile number is correct.
   - In Your Account, visit
     **Login and Security** to confirm your number is correct.
2. Request a new code
   - Select Resend Code.
3. Check that Amazon messages (35213) aren’t blocked
   - On iOS:
     - Go to Settings > Apps > Messaging
     - Ensure SMS is enabled.
     - Navigate to Blocked Contacts and make sure 35213 isn’t listed.
     - Unblock the number if needed, then request the code again.
   - On Android:
     - Go to Apps > Settings > Apps.
     - Tap the three vertical dots and select Special Access.
     - Note: On some devices, this may appear under App list > Filter > Show system apps.
     - Select Use Premium Text Message Services and set to Always allow.
     - If you still don’t receive codes, contact your carrier to enable Short Code SMS.
4. Try again
   - Request a new code and enter it when prompted.
