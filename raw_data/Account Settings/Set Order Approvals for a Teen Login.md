# Set Order Approvals for a Teen Login

You can set order approvals for a teen login in Manage Your Household.

**Note:** As of April 7, 2025, Amazon is no longer accepting new signups for the Teen program.
The information below applies only to existing Teen accounts.

Parents can choose to skip the approval step and set pre-approved spending limits.
You can set order approvals in Manage Your Household.

To set order approvals for a teen login:

1. Go to your Amazon Household.
2. Select Edit Profile for a specific teen.
3. Select Change order approvals.

   Parents will receive itemized notifications for every order, and can cancel and return
   any item in accordance with Amazon's policies.
