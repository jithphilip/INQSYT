# International and Special Item Returns

## International Return Methods

Amazon offers multiple return methods for internationally fulfilled items.

## UPS Drop-Off Returns

Customers shipping from Amazon.com to Canada may receive:
- Prepaid UPS return labels
- Access to UPS drop-off locations

## DHL Express Drop-Off

Available in select countries.

Customers can:
- Drop packages at DHL locations

## DHL Express Pickup

Customers selecting DHL pickup:
- Receive confirmation emails with instructions
- Can schedule pickups through MyDHL+

Pickup scheduling includes:
- Preferred location
- Preferred date
- Preferred time

## Self-Paid Return Labels

Customers using their own shipping labels may receive:
- Reimbursement up to $25 for return postage

Eligibility depends on:
- Country
- Receipt verification

## Heavy and Bulky Item Returns

Heavy or bulky items include products:
- Weighing 50 lbs or more
- Longer than 59 inches
- With girth exceeding 130 inches

Girth formula:
- length + 2 × (width + height)

These returns may:
- Require specialty carriers
- Require scheduled home pickup
- Include additional return shipping fees

## Discounted Purchase Returns

Returning items that qualified for discounts may reduce refund amounts.

## Gift Returns

Gift recipients can initiate returns using:
- 17-digit order numbers
- Packing slips
- Digital gift receipts

If exchanges are needed:
- Customers must return the gift
- Place a new order separately

Kindle books received as gifts may qualify for Amazon Gift Card exchanges before acceptance.

## Bundle Returns

Items purchased in Bundle with Savings promotions:
- Must be returned together
- Do not qualify for partial bundle refunds
